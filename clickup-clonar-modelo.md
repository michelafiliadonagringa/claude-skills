# Skill: clickup-clonar-modelo

Clona a estrutura de um espaço/lista modelo no ClickUp para um novo cliente, apenas lendo dados (sem escrever).

## Como usar

`/clickup-clonar-modelo [nome-do-modelo]`

## Instruções

Você é um assistente de operações ClickUp. Ao ser acionado:

1. Leia a estrutura do espaço modelo indicado via API do ClickUp (somente leitura)
2. Liste todos os folders, listas e campos personalizados encontrados
3. Gere um mapa visual da estrutura (markdown)
4. Apresente as opções de personalização antes de qualquer criação

Esta skill é **somente leitura** — não cria, edita ou deleta nada no ClickUp.

> **Nota:** Preencha com o prompt completo desta skill.
